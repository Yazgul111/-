# Training Planner

**Автор:** Хафизова Язгуль
**Вариант:** "Training Planner"
**Дата:** Май 2026

## Описание

Графическое приложение для планирования тренировок. Позволяет добавлять тренировки с указанием даты, типа и длительности, просматривать их в таблице, фильтровать по типу и диапазону дат. Данные сохраняются в JSON-файл и загружаются при следующем запуске.

## Требования

- Python 3.6 или выше.
- Библиотека Tkinter (входит в стандартную поставку Python).

## Инструкция по запуску

1. Установите Python 3.6+ (если не установлен).
2. Скачайте файлы проекта.
3. Откройте терминал в папке проекта.
4. Запустите:
   python training_planner.py

## Примеры использования

- Добавьте тренировку: введите дату (ГГГГ-ММ-ДД), тип (например, "Кардио"), длительность (например, 60), нажмите «Добавить тренировку».
- Для фильтрации введите тип или даты в соответствующие поля и нажмите «Применить фильтр».
- Для сброса фильтра — «Сбросить фильтр».
- Данные сохраняются в trainings.json.
