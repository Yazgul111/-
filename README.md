# Training Planner

**Автор:** Хафизова Язгуль
**Вариант:** "Training Planner"
**Дата:** Май 2026

## Описание

Графическое приложение для планирования тренировок. Позволяет добавлять тренировки с указанием даты (ГГГГ-ММ-ДД), типа и длительности (минуты), просматривать их в таблице и фильтровать по типу и диапазону дат. Данные сохраняются в файл `trainings.json` и загружаются при запуске.

## Требования

- Python 3.9+
- Библиотека Tkinter (входит в стандартную поставку Python).

## Инструкция по запуску

1. Установите Python 3.9+.
2. Скачайте или клонируйте репозиторий.
3. Откройте терминал в папке проекта.
4. Запустите:
   python training_planner.py

## Примеры использования

1. **Добавление тренировки:**
   - Введите дату: `2026-04-30`
   - Введите тип: `Кардио`
   - Введите длительность: `60`
   - Нажмите «Добавить тренировку».
2. **Фильтрация:**
   - Введите тип: `Сила`
   - Укажите даты: с `2026-04-01` по `2026-04-30`
   - Нажмите «Применить фильтр».
3. **Сброс фильтра:** «Сбросить фильтр».
4. Данные сохраняются автоматически.
